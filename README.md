# 🤖 R Jerphin | Futuristic AI & ML Portfolio Website

Welcome to the official repository for my futuristic, highly interactive, premium personal portfolio website. Designed as a blend of Apple, Tesla, OpenAI, and a cutting-edge AI startup, this portfolio highlights my academic journey, projects, certifications, and internship experiences as an AI & ML Engineer.

🔗 **Live Website Link**: **[https://remijerphin-arch.github.io/Personal-Portfolio/](https://remijerphin-arch.github.io/Personal-Portfolio/)**

---

## 🎨 Design & Aesthetics
- **Dark Theme Vibe**: Pure black background (`#0A0A0A`) with electric blue and neon purple glow accents.
- **Glassmorphism Design**: Frosted glass panels (`backdrop-filter`) with subtle hovering borders and dynamic neon shadows.
- **Micro-interactions**: Custom animations on hover, expandable project panels, typing subheadings, and canvas particle fields.
- **Smooth Cursor Follower**: Lag-compensated glowing radial cursor follower that updates its hue based on the active accent theme.

---

## 🚀 Key Features

### 1. circular Loading screen boot sequence
A cinematic terminal-style loader that simulates system checks, mounts memory allocations from Christ University academic semesters, initializes ISRO propulsion telemetry records, and finishes with a circular percentage progress indicator.

### 2. Local-Storage Profile Avatar Uploader
An interactive profile slot that allows recruiters and visitors to upload a custom profile picture directly in the browser via drag-and-drop or file click. The image is compressed to base64, stored in `localStorage`, and persists across browser refreshes.

### 3. Expandable Projects Showcase
Interactive project cards representing 6 core physical IoT and software systems. Each card includes:
- Stylized vector schematic block diagrams
- Custom technologies tags
- Expandable drawers containing key features list and **challenges solved** (e.g. noise reduction, socket threading, and state lifecycle subscriptions).

### 4. Interactive IDE Resume Previewer
A replica of a modern code editor environment (VS Code style) that displays the structured JSON resume data with line numbering and syntax highlights. Features tabbed navigation for certifications (including L&T and Udemy credentials), clipboard copying, and raw `.json` file download triggers.

### 5. Multi-Accent Theme Selector Matrix 🎨
A floating paint palette widget in the bottom-left corner that opens a grid of **10 different accent presets** (such as Neon Mint, Vaporwave, Sunset Flare, and Crimson Venom). Remaps all primary and secondary glow accents across the entire website instantly.

### 6. Vertical Scrolling Timeline
A chronological path tracking academic progress at Christ University, secondary schooling at St. Joseph's School, and my interim internship at **ISRO Propulsion Complex (IPRC)**.

## 🛠️ Technical Stack
- **Framework**: React (Vite)
- **Styling**: Tailwind CSS (v4) & PostCSS
- **Animations**: Framer Motion & CSS keyframe properties
- **Icons**: Lucide React & inline custom social SVG paths
- **Build Engine**: Rolldown (Vite client bundler)

## 💻 Local Setup & Development

To run this project on your local machine:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/remijerphin-arch/Personal-Portfolio.git
   cd Personal-Portfolio
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
   *(Compiled files will be exported to the `./dist` directory).*
